This project is the frontend for a Sensory-data grabber for Linux (tested on Ubuntu)
The Project currently uses a transparent ssd1306 display wired onto a Raspberry Pico 2.

![IMG_20260203_030510192](https://github.com/user-attachments/assets/c0ec0a2d-1973-4dfd-84f7-31286c57de3c)

Features:
- Display Data sent to device, Format: $:[VALUENAME]# to set headings, values in order to be displayed, seperated by #
- A cute animation featuring a bird flying through town
- A gauge to display RAM usage

The project is intended to be used in conjunction with SensoryBridge Project ( https://github.com/Silberlachs/SensorBridge )

Current state features only a hardcoded RAM-gauge, this is currently on TODO
